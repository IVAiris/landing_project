# landing-form-training

Учебный frontend-проект для гайда «Лендинг и форма».

## Что собираем

Локальный лендинг с секциями:

- hero;
- плюсы;
- вопросы;
- форма.

Форма содержит поля:

- имя;
- email;
- описание задачи.

## Стек и конфигурация

- Next.js;
- React;
- TypeScript;
- npm;
- App Router;
- папка `src`;
- обычный CSS без Tailwind;
- локальный запуск через `npm run dev`.

## Границы урока

В этом проекте пока нет:

- API-роутов;
- серверной отправки формы;
- Zod;
- SQLite;
- CSV;
- авторизации;
- деплоя;
- внешних API.

Сейчас собирается только frontend-слой: страница, якорная навигация, клиентское состояние формы, клиентская проверка и сообщения об ошибке или успехе.
